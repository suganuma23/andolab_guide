このサイトを編集したい場合

このサイトはGitHubで管理されています。内容の追加・修正をしたい場合は、以下の手順でプルリクエストを送ってください。

  1. リポジトリURL: https://github.com/suganuma23/andolab_guide
  2. GitHubでリポジトリをフォークする
  3. フォークしたリポジトリをクローンする 
    git clone https://github.com/自分のユーザー名/andolab_guide.git
  4. 変更を加えてコミット・プッシュする
  5. プルリクエストを送る
  6. 管理者がレビューしてマージします

mainブランチへの直接プッシュはできません。必ずフォーク→プルリクエストの流れで変更を提出してください。
