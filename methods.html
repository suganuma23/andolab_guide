回帰分析 (Regression Analysis)

回帰分析は、データの関係性を数式で表現する手法です。安藤研では、アルゴリズムの計算時間がデータサイズ n に対してどのように増加するかを分析するために使います。

卒業研究では、実験結果をまとめる際に回帰分析を行い、計算量のオーダーを実験的に確認します。

線形回帰（最小二乗法）

最も基本的な回帰分析です。データ点 (x1, y1), ..., (xn, yn) に対して、直線 y = ax + b を当てはめます。

最小二乗法の公式

残差の二乗和 ∑(yi - (axi + b))2 を最小化する a, b は以下で求まる:

a = (n∑xiyi - ∑xi∑yi) / (n∑xi2 - (∑xi)2)

b = (∑yi - a∑xi) / n

決定係数 R2

回帰の当てはまりの良さを表す指標です。1に近いほど良い。

R2 = 1 - SSres / SStot

SStot = ∑(yi - ȳ)2    （全変動）
SSres = ∑(yi - (axi + b))2  （残差変動）
  • R2 = 1: データが完全に直線上にある
  • R2 = 0: 回帰が全く当てはまらない
  • 実験データなら R2 ≥ 0.99 くらいあると良い

対数スケールでの回帰（べき乗則の推定）

アルゴリズムの計算時間は多くの場合、べき乗則に従います:

time = eb · na

この形を直接推定するのは難しいですが、両辺の対数を取ると線形になります:

log(time) = a · log(n) + b

つまり X = log(n), Y = log(time) として線形回帰すれば、傾き a が計算量のオーダーの指数、切片 b が定数係数に対応します。

解釈の例

  • a ≈ 2 → 計算時間は O(n2) 程度
  • a ≈ 3 → 計算時間は O(n3) 程度
  • a ≈ 1 → 計算時間は O(n) 程度

R2 が 1 に近ければ、べき乗則のモデルがデータによく当てはまっていることを意味します。

卒研での実践: make_table.py

卒業研究のプログラム（kSSl_ls.zip）では、experiments/make_table.py で回帰分析を行っています。以下がその流れです。

1. 実験データの収集

各パラメータ（k, l）と各データサイズ n に対して、5つのシード値で実験を行い、time_av（1反復あたりの平均計算時間）を記録します。

# output_kss.csv の形式
k, l, n, time, time_av, func_value
1, 60, 60, 0.2554, 0.0213, 424510654.30
1, 60, 60, 0.5544, 0.0213, 421864245.23
...

2. 各 n での平均を計算

同じ k（または l）と n の組み合わせで、5シード分の time_av を平均します。

3. 対数スケールで線形回帰

log(n) と log(time_av) に対して最小二乗法で回帰します。

# make_table.py の linear_regression 関数
def linear_regression(xs, ys):
    n = len(xs)
    sx  = sum(xs)
    sy  = sum(ys)
    sxx = sum(x * x for x in xs)
    sxy = sum(x * y for x, y in zip(xs, ys))
    a = (n * sxy - sx * sy) / (n * sxx - sx * sx)
    b = (sy - a * sx) / n
    # R^2 の計算
    y_mean = sy / n
    ss_tot = sum((y - y_mean) ** 2 for y in ys)
    ss_res = sum((y - (a * x + b)) ** 2 for x, y in zip(xs, ys))
    r2 = 1.0 - ss_res / ss_tot if ss_tot > 0 else 1.0
    return a, b, r2

ここで xs = [log(60), log(70), log(80), log(90), log(100)]、ys = [log(avg_time) for each n] です。

4. 結果の表にまとめる

各パラメータについて、n ごとの平均計算時間と回帰係数 a, b, R2 を表形式で出力します。
